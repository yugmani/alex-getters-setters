Introduction to JavaScript Property Getters and Setters

Properties and Property Getters and Setters

  • Property getters and setters allow you to change the default behavior when accessing or modifying object properties.

  • Two types of properties in JavaScript:

    1. Data Properties: Used while working with objects.
    2. Accessor Properties: These are simply the methods. These are executed every time working with a property to access or get a value and to set or change a value.
    3. The function that is executed to access some value is getter.
    4. The function that is executed to set or change a value is setter.
    5. These methods are executed automatically, we don't need to call them explicitly.

  • Property Getters

    1. Property getters or methods are used to access (to get value of some property) properties of objects.
    2. This is like a regular object method where we write the 'get' keyword followed by the name of getter method and parentheses.
    3. It is important to make sure that there is a return statement to return something or value in the body of getter method.

  • Property Setters

    1. When you set or change value of some property JavaScript will execute existing property setter, or setter method, for that property.
    2. The keyword 'set' is used, to define setter method.
    3. In addition, we specify at least one parameter in setter method. Specifying is a way to pass that value to a setter so it can be used.
    4. Unlike the getter method, the setter method doesn't have to return anything. Settter is used to set values and no one probably expects it to return anything.
